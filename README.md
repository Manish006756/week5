# Even or Odd Checker

## Overview
This program determines whether a number is even or odd using the modulus operator.

## Functional Requirements
- Accept number input
- Check divisibility by 2
- Display result

## Software Design Principles Applied

### 1. Efficiency
The solution uses a single mathematical condition (`number % 2 == 0`), making it computationally efficient.

### 2. Clarity
The logic is concise and easy to understand, suitable for demonstrating decision-making concepts.

### 3. Reusability
The logic can be reused in larger systems such as number filters, calculators, or validation tools.

### 4. Maintainability
The compact design makes debugging and updating straightforward.

### 5. Correctness
The modulus operator is the most reliable standard method for checking parity.

## Possible Improvements
- Validate numeric input
- Check multiple numbers in loop
- Store results in list or file
